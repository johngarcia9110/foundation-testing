The Grid:

Just like bootstrap, Foundation uses css classes to allow you to quickly create website layouts for your content. Below is some common rules to keep in mind when using Foundation grid classes.

Basics:

In order to use grid columns add an element with the class of .row. Then add elements with the class .column(s). Foundation is mobile first, when sizing your columns go from small-# to medium-# to large-#.
